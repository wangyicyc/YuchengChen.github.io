项目成果展示

多无人机联通遍历搜索

项目描述： 部署分布式多无人机在一个信息地图中进行协同搜索，使无人轨迹分布对齐地图的信息分布，以获取更多信息。实现了基于频域分析的覆盖搜索算法，并通过基于概率的路径规划确保多机连通性。

使用技术： - ROS（机器人操作系统） - iLQR 轨迹优化 - Python/C++ - Gazebo 仿真

风扰动下无人机目标搜索+投放

项目描述： 室外环境下，使无人机载重（一个1.5L矿泉水瓶）在8m80m的跑道上自主飞行，实现以下任务： 1. 无人机在指定起飞点自主起飞 2. 在距离起飞点30m远的85m区域内寻找白桶进行投放，白桶共三个（直径分别为30, 20,15），直径越小分值越高 3. 在距离起飞点65m远的8*5区域内侦查危险物，并远程传送数据，由参赛人员在控制台填写 4. 完成以上任务后无人机自主飞回起飞点并降落

使用技术： - C++ - 模糊 PID 控制 - VFH 路径规划 - 卡尔曼滤波

障碍物森林中的快递无人机

项目描述： 室内环境下，使无人机载重（3个100g快递盒）在1010m的区域执行任务，具体任务如下： 1. 在起飞点自主起飞 2. 在108.5m的区域中寻找存在的5个标靶进行投放，投放得分由投放准确度与标靶自身权重计算，同时注意躲避该区域内的密集障碍物 3. 执行完任务2后，前往10*1.5m的长廊入口，穿越长廊抵达降落点准确降落，其中长廊中存在两个80cm宽的窄道

使用技术： - ROS - FAST-LIO - Ego-Planner - MPC 控制器 - C++

第三届高校智能机器人创意大赛 - 无人机 A 组竞赛

项目描述： 参与无人机 A 组竞赛，专注于具有避障和目标识别功能的自主飞行任务。开发了适用于动态环境的无人机鲁棒飞行控制算法，并利用传感器融合实现实时障碍物检测与避障。

使用技术： - ROS（机器人操作系统） - 计算机视觉 - 传感器融合 - 自主飞行控制 - C++
