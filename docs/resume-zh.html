дёӯж–Үз®ҖеҺҶ

йҷҲе®ҮеҹҺ

йҷҲе®ҮеҹҺпјҢз”·пјҢ2003е№ҙ12жңҲеҮәз”ҹпјҢеҚҺдёӯз§‘жҠҖеӨ§еӯҰжңәжў°з§‘еӯҰдёҺе·ҘзЁӢеӯҰйҷўжңәжў°и®ҫи®ЎеҲ¶йҖ еҸҠе…¶иҮӘеҠЁеҢ–пјҲжңәеҷЁдәәж–№еҗ‘пјүжң¬з§‘еңЁиҜ»пјҢйў„и®Ў2026е№ҙ6жңҲжҜ•дёҡгҖӮе·ІдҝқйҖҒиҮідёҠжө·дәӨйҖҡеӨ§еӯҰжҺ§еҲ¶з§‘еӯҰдёҺе·ҘзЁӢдё“дёҡж”»иҜ»зЎ•еЈ«з ”з©¶з”ҹпјҢе°ҶдәҺ2026е№ҙ9жңҲе…ҘеӯҰгҖӮ

е®һд№ з»ҸеҺҶ

йҖҹи…ҫиҒҡеҲӣ(Robosense)

  • ејәеҢ–еӯҰд№ з®—жі•е®һд№ з”ҹ | 2025.12.10 - иҮід»Ҡ
  • йЎ№зӣ®жҸҸиҝ°пјҡзҪ‘зҗғжҺҘжү“е°ҸиҪҰзҡ„ејәеҢ–еӯҰд№ жЁЎеһӢи®ӯз»ғдёҺйғЁзҪІ
  • дё»иҰҒе·ҘдҪңпјҡ
    1. ејәеҢ–еӯҰд№ жЁЎеһӢи®ӯз»ғпјҡеңЁIsaac Gymи®ӯз»ғзҺҜеўғдёӢпјҢеҹәдәҺrsl_rlејәеҢ–еӯҰд№ еә“е®һзҺ°PPOз®—жі•пјҢи®ӯз»ғе°ҸиҪҰе®ҢжҲҗзҪ‘зҗғжҺҘжү“иҮӘеҠЁжҺ§еҲ¶д»»еҠЎгҖӮ
    2. Sim2Sim Gapз ”з©¶пјҡе°ҶжЁЎеһӢд»ҺIsaac Gymи®ӯз»ғзҺҜеўғиҝҒз§»еҲ°Mujocoд»ҝзңҹзҺҜеўғпјҢйӘҢиҜҒжЁЎеһӢеңЁдёҚеҗҢд»ҝзңҹзҺҜеўғдёӢзҡ„иЎЁзҺ°пјҢе®һзҺ°и·ЁзҺҜеўғжіӣеҢ–иғҪеҠӣгҖӮ
    3. Real2Sim Gapз ”з©¶пјҡе°ҶжЁЎеһӢд»ҺиҷҡжӢҹи®ӯз»ғзҺҜеўғйғЁзҪІеҲ°зңҹе®һзү©зҗҶзҺҜеўғе№¶иҝӣиЎҢйӘҢиҜҒпјҢйӘҢиҜҒжЁЎеһӢеңЁзңҹе®һзҺҜеўғдёӢзҡ„иЎЁзҺ°гҖӮ
    4. еҘ–еҠұеҮҪж•°и®ҫи®ЎпјҡеҹәдәҺзңҹе®һзҺҜеўғеҸҚйҰҲи®ҫи®ЎеҘ–еҠұеҮҪж•°пјҢеј•еҜјжЁЎеһӢеӯҰд№ жҺ§еҲ¶зӯ–з•ҘпјҢжҸҗеҚҮе°ҸиҪҰзҡ„жҺҘзҗғжҲҗеҠҹзҺҮгҖӮ

дё»иҰҒйЎ№зӣ®з»ҸеҺҶ

Actor-Criticз®—жі• + MPCжҺ§еҲ¶еҷЁе®һзҺ°ж— дәәжңәиҮӘдё»йЈһиЎҢ

йЎ№зӣ®иҙҹиҙЈдәәпҪң 2026.02 вҖ“ 2026.06
йЎ№зӣ®жҸҸиҝ°пјҡз»“еҗҲејәеҢ–еӯҰд№ дёҺжЁЎеһӢйў„жөӢжҺ§еҲ¶жҠҖжңҜпјҢжҸҗеҚҮж— дәәжңәеңЁеӨҚжқӮзҺҜеўғдёӢзҡ„йЈһиЎҢйІҒжЈ’жҖ§пјҢе°Өе…¶й’ҲеҜ№е®ҡдҪҚдҝЎжҒҜеҸ—е№Іжү°зҡ„еңәжҷҜ
жң¬дәәиҙҹиҙЈд»»еҠЎпјҡ
1. з®—жі•иһҚеҗҲпјҡе°ҶActor-CriticејәеҢ–еӯҰд№ з®—жі•дёҺMPCжҺ§еҲ¶еҷЁзӣёз»“еҗҲпјҢеҲ©з”ЁејәеҢ–еӯҰд№ иҮӘйҖӮеә”и°ғж•ҙMPCжҺ§еҲ¶зӯ–з•Ҙзҡ„д»Јд»·еҮҪж•°пјҢеўһејәзі»з»ҹеҜ№зҺҜеўғеҸҳеҢ–зҡ„йҖӮеә”иғҪеҠӣ
2. еҸҜеҫ®еҲҶдјҳеҢ–пјҡејҖеҸ‘еҹәдәҺеҸҜеҫ®еҲҶдјҳеҢ–зҡ„жҺ§еҲ¶жЎҶжһ¶пјҢе°ҶMPCзҡ„дјҳеҢ–иҝҮзЁӢзәіе…ҘејәеҢ–еӯҰд№ зҡ„и®ӯз»ғжөҒзЁӢдёӯ
3. жЁЎеһӢи®ӯз»ғпјҡйҮҮз”ЁPPOз®—жі•иҝӣиЎҢејәеҢ–еӯҰд№ и®ӯз»ғпјҢеҲҶеҲ«дҪҝз”ЁMLPе’ҢTransformerзҪ‘з»ңз»“жһ„жһ„е»әActor-CriticжЁЎеһӢ

йЈҺжү°еҠЁдёӢж— дәәжңәзӣ®ж Үжҗңзҙў+жҠ•ж”ҫ

йЎ№зӣ®иҙҹиҙЈдәәпҪң2024.05 вҖ“ 2024.10
йЎ№зӣ®жҸҸиҝ°пјҡе®ӨеӨ–зҺҜеўғдёӢпјҢдҪҝж— дәәжңәиҪҪйҮҚпјҲдёҖдёӘ1.5Lзҹҝжіүж°ҙз“¶пјүеңЁ8m80mзҡ„и·‘йҒ“дёҠиҮӘдё»йЈһиЎҢпјҢе®һзҺ°д»ҘдёӢд»»еҠЎпјҡ
1. ж— дәәжңәеңЁжҢҮе®ҡиө·йЈһзӮ№иҮӘдё»иө·йЈһ
2. еңЁи·қзҰ»иө·йЈһзӮ№30mиҝңзҡ„85mеҢәеҹҹеҶ…еҜ»жүҫзҷҪжЎ¶иҝӣиЎҢжҠ•ж”ҫпјҢзҷҪжЎ¶е…ұдёүдёӘпјҲзӣҙеҫ„еҲҶеҲ«дёә30, 20,15пјүпјҢзӣҙеҫ„и¶Ҡе°ҸеҲҶеҖји¶Ҡй«ҳ
3. еңЁи·қзҰ»иө·йЈһзӮ№65mиҝңзҡ„8*5еҢәеҹҹеҶ…дҫҰжҹҘеҚұйҷ©зү©пјҢе№¶иҝңзЁӢдј йҖҒж•°жҚ®пјҢз”ұеҸӮиөӣдәәе‘ҳеңЁжҺ§еҲ¶еҸ°еЎ«еҶҷ
4. е®ҢжҲҗд»ҘдёҠд»»еҠЎеҗҺж— дәәжңәиҮӘдё»йЈһеӣһиө·йЈһзӮ№е№¶йҷҚиҗҪ

жң¬дәәиҙҹиҙЈд»»еҠЎпјҡ
1. жЁЎзіҠPIDжҺ§еҲ¶еҷЁзј–еҶҷ(C++)пјҡйҖҡиҝҮжЁЎзіҠи§„еҲҷвҫғйҖӮеә”и°ғж•ҙPIDзі»ж•°пјҢе®һзҺ°в»ӣжү°еҠЁдёӢвҪҶвјҲжңәзІҫеҮҶжҺ§еҲ¶
2. зј–еҶҷи·Ҝеҫ„и§„еҲ’з®—жі•VFHпјҡдҫқжҚ®VFHз®—жі•жұӮи§ЈеҪ“еүҚеҸҜвҫҸзӮ№пјҢе№¶йҖҡиҝҮBж ·жқЎжӣІзәҝдјҳеҢ–еҮәе…үж»‘и·Ҝеҫ„
3. йғЁзҪІж»Өжіўз®—жі•пјҡйҖҡиҝҮеҚЎе°”жӣјж»Өжіўд»ҘеҸҠдҪҺйҖҡж»ӨжіўеҷЁпјҢе®һзҺ°еҜ№еҷӘеЈ°зҡ„ж»ӨйҷӨдҪңз”ЁпјҢдҪҝжҺ§еҲ¶еҷЁе№іж»‘иҫ“еҮә

йҡңзўҚзү©жЈ®жһ—дёӯзҡ„еҝ«йҖ’ж— дәәжңә

йЎ№зӣ®иҙҹиҙЈдәәпҪң2023.10 вҖ“ 2024.05
йЎ№зӣ®жҸҸиҝ°пјҡе®ӨеҶ…зҺҜеўғдёӢпјҢдҪҝж— дәәжңәиҪҪйҮҚпјҲ3дёӘ100gеҝ«йҖ’зӣ’пјүеңЁ1010mзҡ„еҢәеҹҹжү§иЎҢд»»еҠЎпјҢе…·дҪ“д»»еҠЎеҰӮдёӢпјҡ
1. еңЁиө·йЈһзӮ№иҮӘдё»иө·йЈһ
2. еңЁ108.5mзҡ„еҢәеҹҹдёӯеҜ»жүҫеӯҳеңЁзҡ„5дёӘж Үйқ¶иҝӣиЎҢжҠ•ж”ҫпјҢжҠ•ж”ҫеҫ—еҲҶз”ұжҠ•ж”ҫеҮҶзЎ®еәҰдёҺж Үйқ¶иҮӘиә«жқғйҮҚи®Ўз®—пјҢеҗҢж—¶жіЁж„ҸиәІйҒҝиҜҘеҢәеҹҹеҶ…зҡ„еҜҶйӣҶйҡңзўҚзү©
3. жү§иЎҢе®Ңд»»еҠЎ2еҗҺпјҢеүҚеҫҖ10*1.5mзҡ„й•ҝе»Ҡе…ҘеҸЈпјҢз©ҝи¶Ҡй•ҝе»ҠжҠөиҫҫйҷҚиҗҪзӮ№еҮҶзЎ®йҷҚиҗҪпјҢе…¶дёӯй•ҝе»ҠдёӯеӯҳеңЁдёӨдёӘ80cmе®Ҫзҡ„зӘ„йҒ“

жң¬дәәиҙҹиҙЈд»»еҠЎпјҡ
1. UbuntuдёӢзј–еҶҷеҹәдәҺROSзҡ„в»ңвҫҸд»Јз Ғ(C++)пјҡжү§вҫҸйҡңзўҚжЈ®жһ—гҖҒзӢӯзӘ„вҫӣе»ҠзҺҜеўғдёӢйҒҝйҡңд»»еҠЎеҸҠжҠ•ж”ҫд»»еҠЎ
2. йғЁзҪІFAST-LIOеҸҠйҒҝйҡңз®—жі•Ego-Planner : вҪҶвјҲжңәе®ҡдҪҚе№¶вҪЈжҲҗеҸҜвҫҸи·ҜзәҝпјҢMPCжҺ§еҲ¶еҷЁи·ҹиёӘи·Ҝеҫ„йҒҝйҡң
3. зј–еҶҷMPCжҺ§еҲ¶еҷЁ(C++)пјҡдҫқжҚ®зӣ®ж ҮзӮ№дёҺзәҰжқҹжқЎд»¶еңЁзәҝи®Ўз®—еҜ№вҪҶвјҲжңәеҠ йҖҹеәҰзҡ„жҺ§еҲ¶пјҢд»Ҙи·ҹиёӘе…ЁеұҖи·Ҝеҫ„

  • и·Ҝеҫ„и§„еҲ’
  • ејәеҢ–еӯҰд№ 

дё“дёҡжҠҖиғҪ

  • жңәеҷЁдәәж“ҚдҪңзі»з»ҹпјҡзҶҹз»ғдҪҝз”Ё ROSпјҢжҺҢжҸЎ Ubuntu е‘Ҫд»ӨиЎҢеҸҠ Gazebo д»ҝзңҹ
  • ејәеҢ–еӯҰд№ з®—жі•пјҡзҶҹжӮү Actor-CriticгҖҒTRPOгҖҒPPOгҖҒSACгҖҒDDPG зӯү
  • жңәеҷЁдәәеӯҰеҹәзЎҖпјҡжҺҢжҸЎ IKпјҲйҖҶиҝҗеҠЁеӯҰпјүгҖҒж—ӢиҪ¬зҹ©йҳөгҖҒеӣӣе…ғж•°зӯү
  • жңҖдјҳжҺ§еҲ¶ж–№жі•пјҡзҶҹжӮү LQRгҖҒMPCгҖҒiLQRгҖҒDDP зӯүжҺ§еҲ¶еҷЁ
